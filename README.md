
## 基于vue表格和变表单组件

主要配合[pcadmin](https://github.com/ksc-fx/pcadmin)使用。
使用方法：
```
npm install pcadmin-select
```

```javascript

import select from 'pcadmin-select';

```

### Data
| 参数      | 说明          | 类型      | 是否必选                           | 可选值  | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |-------- |
| v-model | 绑定数据 | String | Number | - |  |
| options | 选择项 | Atrray | - |  |


### Events
| 事件名称 | 说明 | 回调参数 |
|---------- |-------- |---------- |
| selectchange | 点击回调函数 | — |
